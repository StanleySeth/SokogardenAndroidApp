<b>1. What is Loopj in Android?</b> <b>(Android Asynchronous HTTP Client)</b>It refers to the library used in Android to perform asynchronous HTTP requests. It is available on Maven Central <br>
<b>Key Points</b><br>1. Used for <b>GET, POST, PUT, DELETE</b><br>2. Runs in <b>background thread</b>.<br>3. Prevents <b>UI Freezing(ANR).</b><br>4. Uses <b>callbacks (onSuccesss, onFailure).</b><br>
<b>Advantages</b><br>1. Simple to use</b><br>2. Handles networking easily</b><br>3. No need to manage threads manually</b><br>
<b>Disadvantages</b><br>1. Outdated/deprecated</b><br>2. Uses old Apache HTTP APIs</b><br>3. Not recommended for modern apps</b><br><br>

<b>What is SharedPreferences?</b>
SharedPreferences is an Android API used to store <b>persistent key-value data</b> in a local XML file within the app’s internal storage.
